\section{Background and Objective}
\label{sec:background}

\subsection{From closed-set segmentation to open-vocabulary segmentation}
\label{subsec:closed_to_ovs}

Image segmentation has long been studied as a dense prediction problem under a closed-set formulation. 
In this setting, the semantic categories to be predicted are predefined during training and remain unchanged during evaluation. 
Representative tasks include semantic segmentation, instance segmentation, and panoptic segmentation. 
Semantic segmentation assigns a category label to each pixel, thereby providing a dense scene-level parsing result. 
Instance segmentation further distinguishes different object instances within the same category by predicting object-level masks. 
Panoptic segmentation unifies semantic-level stuff prediction and instance-level thing prediction, thus offering a more complete interpretation of a visual scene \cite{Long2015,He2017,Kirillov2019}.

Although these task formulations have substantially advanced pixel-level visual understanding, they share a common assumption: the model is optimized for a fixed training taxonomy. 
This assumption makes the evaluation protocol clear and controllable, since both the training and testing categories are defined in advance. 
However, it also restricts the applicability of segmentation models in open-world scenarios. 
A model trained on a fixed label space may fail when the target category is unseen, when the object belongs to a long-tail concept, or when the user describes the target with a category name that is absent from the training annotation vocabulary. 
Therefore, the central limitation of closed-set segmentation is not only the lack of category coverage, but also the lack of flexibility in how the target concept can be specified.

Open-vocabulary segmentation relaxes this fixed-category assumption by allowing the target label space to be defined at inference time. 
Instead of restricting prediction to a closed set of training categories, open-vocabulary segmentation uses category names, text prompts, or other language descriptions as semantic queries. 
The segmentation model is therefore expected to associate visual regions with text-defined concepts, including concepts that may not have appeared in the segmentation training set. 
This formulation changes the role of language in segmentation: language is no longer only used for dataset annotation, but becomes an explicit interface for specifying the target vocabulary.

This transition is closely related to the development of vision-language foundation models. 
Models such as CLIP learn transferable image-text alignment from large-scale image-text pairs and make it possible to compare visual regions with textual concepts in a shared representation space \cite{Radford2021}. 
Based on this capability, open-vocabulary segmentation methods attempt to transfer image-level semantic alignment to pixel-level or mask-level prediction. 
For example, CLIP-based methods commonly combine class-agnostic mask proposals, region-level visual features, and text embeddings to recognize categories beyond the training vocabulary \cite{Liang2023,Xu2023b}. 
However, this transfer is not straightforward, because image-text foundation models are usually trained for global image recognition rather than dense localization.

The key challenge in open-vocabulary segmentation therefore lies in balancing semantic openness and spatial precision. 
Text-aligned visual representations provide strong open-vocabulary recognition ability, but they may not preserve fine-grained object boundaries or pixel-level spatial details. 
Conversely, classical segmentation models provide accurate dense prediction, but their semantic space is typically tied to the annotated categories used during training. 
As a result, modern open-vocabulary segmentation methods often combine language-aligned semantic representations with segmentation-specific mask generation or dense decoding modules. 
This tension between open semantic recognition and accurate spatial localization forms the basic technical motivation for the method families reviewed in the following sections.

\subsection{Task formulations in language-guided segmentation}
\label{subsec:language_guided_tasks}

Language-guided segmentation further extends the role of text in dense prediction. 
In closed-set segmentation, language is mainly reflected in the predefined class names used for dataset annotation. 
In open-vocabulary and reasoning segmentation, however, language becomes part of the model input and directly specifies, describes, or implies the target to be segmented. 
According to the form of the language input and the ambiguity of the target object, existing tasks can be broadly organized into open-vocabulary segmentation, referring segmentation, and reasoning segmentation.

Open-vocabulary segmentation mainly focuses on text-defined category recognition. 
Given an image and a set of category names or text prompts, the model is required to assign pixels or masks to the corresponding textual categories. 
The input language is usually explicit and category-level, such as ``dog'', ``road'', or ``traffic light''. 
Therefore, the key question is whether the model can transfer semantic knowledge from vision-language pretraining to dense prediction, especially when the target categories are absent from the segmentation training set. 
This task formulation is closely related to CLIP-based segmentation methods, where text embeddings are used as open-vocabulary classifiers for visual regions or masks \cite{Radford2021,Liang2023,Xu2023b}.

Referring segmentation introduces a more instance-specific form of language supervision. 
Different from open-vocabulary segmentation, where the input is often a category name, referring segmentation uses natural-language expressions to identify a particular object instance in the image. 
For example, a phrase such as ``the man wearing a red shirt'' does not merely specify the category ``man'', but also uses visual attributes and contextual relations to distinguish one instance from others. 
Thus, referring segmentation requires the model to align linguistic descriptions with object-level visual evidence. 
The target is usually explicit in the expression, but the model must resolve attributes, spatial relations, and instance-level distinctions before generating the final mask.

Reasoning segmentation further increases the complexity of the language input. 
In this setting, the target object may not be explicitly named by a category label or a direct referring expression. 
Instead, it must be inferred from complex instructions, commonsense knowledge, visual relations, or scene context. 
For example, a query such as ``the object that the person will use to cut the paper'' requires the model to understand the action, infer the likely object, and then localize it at the pixel level. 
Therefore, reasoning segmentation is not only a segmentation problem, but also a joint problem of language understanding, visual reasoning, and mask prediction.

The distinction among these tasks can be understood from the perspective of language complexity and target ambiguity. 
Open-vocabulary segmentation mainly changes the label space from fixed categories to text-defined categories. 
Referring segmentation changes the target from category-level regions to expression-specified object instances. 
Reasoning segmentation further changes the input from explicit descriptions to implicit instructions that require inference. 
As the task moves along this direction, the segmentation system must rely less on predefined category taxonomies and more on general vision-language understanding.

This task hierarchy also explains why different model families are needed. 
CLIP-based methods are naturally suited to open-vocabulary category transfer, because they provide strong image-text alignment. 
However, they are not originally designed for fine-grained localization or complex reasoning. 
Promptable segmentation models such as SAM provide strong class-agnostic mask priors, but they do not directly solve text-based semantic recognition \cite{Kirillov2023}. 
MLLM-based methods, such as LISA, are introduced to bridge this gap by using language-model reasoning ability to interpret complex instructions and connect them with mask prediction \cite{Lai2024}. 
Therefore, the evolution from open-vocabulary segmentation to reasoning segmentation reflects a broader shift from text-defined recognition toward language-guided visual reasoning.

\subsection{Pixel grounding and multimodal reasoning}
\label{subsec:pixel_grounding}

The transition from open-vocabulary segmentation to reasoning segmentation also changes the role of segmentation outputs. 
In conventional segmentation tasks, the output is usually evaluated as a pixel-wise label map or a set of object masks. 
In language-guided and multimodal settings, however, the mask is no longer only a dense prediction result. 
It becomes a grounding interface that connects visual regions with linguistic units, such as category names, referring expressions, instruction tokens, generated phrases, or conversational responses. 
This shift motivates the notion of pixel grounding, where language understanding must be associated with spatially precise mask prediction.

Pixel grounding emphasizes the correspondence between language and image regions at the pixel level. 
A grounded segmentation model should not only recognize that a textual concept is present in the image, but also localize the corresponding visual evidence with an accurate mask. 
For example, when a model generates or receives a phrase such as ``the broken chair near the window'', it should identify the object implied by the phrase and produce the pixel region that supports this expression. 
Therefore, pixel grounding requires semantic alignment, instance disambiguation, and spatial localization to be jointly considered. 
This makes it different from image-level vision-language alignment, where the model only needs to match a whole image with a caption or a text prompt.

Multimodal large language models further extend this formulation by introducing free-form language understanding and generation into segmentation. 
Instead of mapping a fixed set of textual labels to masks, MLLM-based systems can process complex instructions, answer visual questions, and produce language responses conditioned on the image. 
To make these responses spatially grounded, the model must connect language representations inside the MLLM with mask prediction modules. 
A common design is to introduce special tokens or phrase-level representations that serve as bridges between the language model and a segmentation decoder. 
In this way, the language model provides high-level semantic and reasoning information, while the segmentation module converts this information into pixel-level masks.

LISA is a representative example of this direction. 
It formulates reasoning segmentation with large language models and introduces a special segmentation token to connect the output of the language model with mask generation \cite{Lai2024}. 
The model first interprets the complex instruction through the MLLM, and the hidden representation associated with the segmentation token is then used to guide the mask decoder. 
This design allows the system to segment targets that are not directly specified by category names, but must be inferred from the instruction and the visual context. 
Thus, LISA illustrates how reasoning ability and segmentation ability can be coupled through a language-to-mask interface.

Grounded multimodal conversation further broadens the problem. 
In this setting, the model is not limited to producing a single mask for a single instruction. 
Instead, it may generate a natural-language response in which multiple phrases or entities are linked to corresponding pixel-level masks. 
GLaMM follows this direction by associating generated phrase-level expressions with visual masks, thereby turning pixel grounding into part of a multimodal dialogue system \cite{Rasheed2024}. 
Compared with single-target reasoning segmentation, grounded conversation requires the model to maintain consistency between generated language, visual entities, and mask-level localization.

This development also introduces new challenges. 
First, language correctness and mask correctness are not equivalent. 
A model may generate a plausible answer while grounding it to the wrong region, or it may localize an object correctly while misunderstanding the reasoning query. 
Second, pixel grounding requires fine-grained spatial precision, whereas MLLMs are mainly optimized for high-level semantic reasoning and language generation. 
Third, evaluation becomes more complex because the system must be assessed not only by segmentation metrics such as IoU, but also by whether the generated language is faithful to the image and whether the grounded masks correctly support the response. 
These challenges indicate that reasoning segmentation and pixel grounding should be analyzed as joint problems of language understanding, visual reasoning, and dense spatial prediction.

Therefore, pixel grounding provides an important conceptual bridge between open-vocabulary segmentation and multimodal reasoning systems. 
Open-vocabulary segmentation focuses on recognizing text-defined categories, while reasoning segmentation requires the model to infer implicit targets from complex language. 
Pixel grounding further asks whether the model can make its language understanding spatially explicit at the mask level. 
This perspective is central to the later comparison of MLLM-based reasoning segmentation, grounded conversation, and future extensions to video, 3D, and multi-context visual grounding.

\subsection{Survey scope and comparison dimensions}
\label{subsec:scope_dimensions}

Based on the above discussion, this survey focuses on open-vocabulary and reasoning segmentation with vision-language foundation models. 
The main scope covers methods that use language as an explicit semantic interface for dense visual prediction, including category-name-based open-vocabulary segmentation, promptable segmentation, unified vision-language decoding, reasoning segmentation, and pixel grounding. 
Although classical closed-set segmentation provides the historical and technical background, it is not the main comparison target of this survey. 
Instead, closed-set segmentation is used as the baseline formulation from which open-vocabulary and reasoning-oriented segmentation tasks have evolved.

The reviewed methods are organized according to technical routes rather than a purely chronological order. 
Specifically, we discuss CLIP-based open-vocabulary segmentation, diffusion-based open-vocabulary segmentation, SAM-based promptable and open-vocabulary segmentation, unified vision-language segmentation models, and MLLM-based reasoning segmentation and pixel grounding. 
This organization is adopted because these method families differ not only in model architecture, but also in task assumption, input form, supervision type, output representation, and evaluation protocol. 
A route-based taxonomy can therefore better reveal the conceptual transition from text-defined category recognition to language-guided reasoning and grounded multimodal interaction.

Meanwhile, this survey keeps a bounded scope. 
The primary focus is on image-level segmentation, where the input is a single image and the output is a semantic map, object mask, class-aware mask, or phrase-grounded mask. 
Video reasoning segmentation, 3D grounding, multiview segmentation, and multi-context visual grounding are treated as extension directions rather than the central body of the survey. 
This choice allows the main discussion to remain focused on the interaction between vision-language foundation models and dense mask prediction, while still showing how the same technical challenges may extend to more complex spatial-temporal settings.
